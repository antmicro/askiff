# Usage Highlights

## Typing oriented

* Strongly recommended to use with type checkers
* Ensuring type correctness is necessary for correct serialization

## Enum based values

* It is recommended to operate on provided enums than on raw values

## See

* PCB Layer:
  * enum & inheritance based
  * See {py:class}`askiff.common_pcb.BaseLayer` & {py:class}`askiff.common_pcb.LayerSet`
* Askiff Project:
  * Unified entry point for project
  * See {py:class}`askiff.AskiffPro`
* Property list
  * See {py:class}`askiff.common.PropertyList`
* GrItem & GrShape
  * See {py:class}`askiff.gritems.GrItem` & {py:class}`askiff.gritems.GrShape`
* BBox
  * See {py:class}`askiff.common.BBox`
